# R-splitters

A package for grouping and splitting dataframes and vectors in R.  

By Ludvig R. Olsen
Cognitive Science, Aarhus University  
Started in Oct. 2016  

Main functions:  
* group_factor  
* group 
* splt  

  
Please read using_splitters.pdf for further information on how to use the functions.
